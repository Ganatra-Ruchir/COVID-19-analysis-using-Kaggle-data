# Contributing

Thank you for considering a contribution to the COVID Streamlit Dashboard project.

## How to contribute

1. Fork the repository.
2. Create a feature branch from `main`.
3. Make your changes and keep commits small and focused.
4. Submit a pull request with a clear summary of the change.

## Code style

- Use meaningful variable and function names.
- Keep Python files formatted consistently.
- Keep notebook content minimal and focused on analysis or visualization.

## Issue reporting

When reporting an issue, include:

- A clear description of the problem.
- Steps to reproduce.
- Expected versus actual behavior.
- Any relevant screenshots or logs.

## Pull request expectations

- Provide a short summary of the change.
- Include any testing or validation steps.
- Reference an issue if applicable.
- Do not merge your own pull request without review if the change affects core behavior.
